Introduction to Image segmentation

Image segmentation is the process by which you label each pixel to belong to a certain type. This way you group pixel belonging to specific objects that you are looking for in an image. The purpose of this article is look at technologies used to segment humans in a picture. This way you can place the segmented human being into a different environment or to decorate him with a google or a hat. In studios recordings are made against green screen to be replaced with a different background during broadcast. This is also referred to as chroma key encoding.

Video: 5 creative green screen effects !


The chroma key encoding required quite a bit of setup and is usually done in a studio. With image segmentation technologies maturing you need to ask today whether you really need that green screen ?

The image segmentation today is being done using deep learning models. We will explore some of the current research that is going on in this domain. First lets look at the top deep learning models that that exists today. They are pytorch and tensorflow. These are defined as follows, on their respective websites as:

  • Pytorch

    An open source machine learning framework that accelerates the path from research prototyping to production deployment.

    PyTorch is an open source machine learning library based on the Torch library, used for applications such as computer vision and natural language processing, primarily developed by Facebook's AI Research lab (FAIR). It is free and open-source software released under the Modified BSD license. Although the Python interface is more polished and the primary focus of development, PyTorch also has a C++ interface.

  • Tensorflow

    TensorFlow is an end-to-end open source platform for machine learning. It has a comprehensive, flexible ecosystem of tools, libraries and community resources that lets researchers push the state-of-the-art in ML and developers easily build and deploy ML powered applications.

    TensorFlow is a free and open-source software library for machine learning. It can be used across a range of tasks but has a particular focus on training and inference of deep neural networks. Tensorflow is a symbolic math library based on dataflow and differentiable programming. It is used for both research and production at Google. TensorFlow was developed by the Google Brain team for internal Google use. It was released under the Apache License 2.0 in 2015.

The top two deep learning platforms are from Facebook and Google and overtime their features are converging. Mircosoft has come up with its own deep learning platform. The main contribution though is to define a framework for machine learning model called ONNX. ONNX is an open format built to represent machine learning models. ONNX defines a common set of operators - the building blocks of machine learning and deep learning models - and a common file format to enable AI developers to use models with a variety of frameworks, tools, runtimes, and compilers.

Theoretically it is possible to convert from a pytroch ckpt model to onnx to tensorflow model. During my last try made in Dec 2020, I was able to convert pytorch model to ONNX but several attempts to convert it to tensorflow model failed.
