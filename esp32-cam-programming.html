ESP 32 CAM Programming


The ESP32-CAM AI-Thinker development board can be programmed using its serial ports. This guide shows how to program and upload code to the ESP32-CAM (AI-Thinker) development board using TTL or FTDI cable. You can also program it using a Arduino UNO board.
